# 🌤️ Raximov Weather Bot

Ushbu Telegram bot foydalanuvchilarga istalgan shahar yoki hududdagi eng soʻnggi ob-havo maʼlumotlarini taqdim etadi. Bot Python tilida yozilgan boʻlib, sodda va tez ishlashga moʻljallangan.

## 🚀 Xususiyatlari

* 🌍 **Istalgan hudud ob-havosi:** Shahar nomini kiritish orqali joriy ob-havoni bilish.
* 🌡️ **Batafsil maʼlumot:** Harorat, namlik, shamol tezligi va ob-havo holati (bulutli, quyoshli va h.k.).
* ⚡ **Tezkor javob:** Foydalanuvchi soʻrovlariga soniyalar ichida javob berish.

---

## 🛠️ Texnologiyalar

Loyiha quyidagi texnologiyalar yordamida yaratilgan:

* **Dasturlash tili:** Python 3.x
* **Kutubxonalar:** `aiogram` (yoki `pyTelegramBotAPI`), `requests`
* **API:** OpenWeatherMap API (ob-havo maʼlumotlarini olish uchun)
